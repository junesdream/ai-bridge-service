# 🤖 AI-Bridge-Service
An enterprise-ready FastAPI backend designed for LLM orchestration and AI-driven insights.

## ✨ Features

| Feature | Description |
|---|---|
| ⚡ **FastAPI Core** | High-performance asynchronous API framework |
| 🔒 **Pydantic Validation** | Strict data typing for reliable request handling |
| 🧩 **Scalable Architecture** | Modular approach to easily swap AI providers |
| 🧪 **Mock-LLM Support** | Integrated fallback logic for robust development cycles |

## 🛠️ Tech Stack

- 🐍 **Python** 3.14
- 🚀 **FastAPI**
- 🦄 **Uvicorn**
- ✅ **Pydantic**

## 🚀 How to Run

**1. Install dependencies**
```bash
pip install -r requirements.txt
```

**2. Start the server**
```bash
python main.py
```

**3. Open interactive API docs**

Navigate to 👉 [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
